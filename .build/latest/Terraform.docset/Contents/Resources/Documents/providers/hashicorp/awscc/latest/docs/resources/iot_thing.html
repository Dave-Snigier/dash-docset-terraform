awscc_iot_thing (Resource)

Resource Type definition for AWS::IoT::Thing

Example Usage

resource "awscc_iot_thing" "example" {
  thing_name = "example"
  attribute_payload = {
    attributes = {
      name = "examplevalue"
    }
  }
}

Schema

Optional

  • attribute_payload (Attributes) (see below for nested schema)
  • thing_name (String)

Read-Only

  • arn (String)
  • id (String) Uniquely identifies the resource.
  • thing_id (String)

Nested Schema for attribute_payload

Optional:

  • attributes (Map of String)

Import

Import is supported using the following syntax:

$ terraform import awscc_iot_thing.example "thing_name"
