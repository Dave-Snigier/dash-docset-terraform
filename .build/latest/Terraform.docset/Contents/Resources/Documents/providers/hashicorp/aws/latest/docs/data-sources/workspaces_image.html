Data Source: aws_workspaces_image

Use this data source to get information about a Workspaces image.

Example Usage

data aws_workspaces_image example {
  image_id = "wsi-ten5h0y19"
}

Argument Reference

This data source supports the following arguments:

  • image_id – (Required) ID of the image.

Attribute Reference

This data source exports the following attributes in addition to the arguments above:

  • name – The name of the image.
  • description – The description of the image.
  • os – The operating system that the image is running.
  • required_tenancy – Specifies whether the image is running on dedicated hardware. When Bring Your Own License (BYOL) is enabled, this value is set to DEDICATED. For more information, see Bring Your Own Windows Desktop Images.
  • state – The status of the image.
