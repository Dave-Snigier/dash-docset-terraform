awscc_wisdom_assistant (Resource)

Definition of AWS::Wisdom::Assistant Resource Type

Example Usage

resource "awscc_wisdom_assistant" "example" {
  name        = "example"
  description = "Example assistant"
  type        = "AGENT"
  server_side_encryption_configuration = {
    kms_key_id = awscc_kms_key.example.arn
  }

  tags = [{
    key   = "ModifiedBy"
    value = "AWSCC"
  }]

}

Schema

Required

  • name (String)
  • type (String)

Optional

  • description (String)
  • server_side_encryption_configuration (Attributes) (see below for nested schema)
  • tags (Attributes Set) (see below for nested schema)

Read-Only

  • assistant_arn (String)
  • assistant_id (String)
  • id (String) Uniquely identifies the resource.

Optional:

Nested Schema for tags

Optional:

  • key (String)
