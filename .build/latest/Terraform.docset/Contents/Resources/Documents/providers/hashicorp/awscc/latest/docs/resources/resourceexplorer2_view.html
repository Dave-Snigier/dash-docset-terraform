awscc_resourceexplorer2_view (Resource)

Definition of AWS::ResourceExplorer2::View Resource Type

Example Usage

resource "awscc_resourceexplorer2_view" "example" {
  view_name = "example"
  included_properties = [
    {
      name = "tags"
    }
  ]

  depends_on = [awscc_resourceexplorer2_index.example]
}

resource "awscc_resourceexplorer2_index" "example" {
  type = "LOCAL"
}

Schema

Required

  • view_name (String)

Optional

  • filters (Attributes) (see below for nested schema)
  • included_properties (Attributes List) (see below for nested schema)
  • scope (String)
  • tags (Map of String)

Read-Only

  • id (String) Uniquely identifies the resource.
  • view_arn (String)

Nested Schema for filters

Optional:

  • filter_string (String)

Nested Schema for included_properties

Optional:

  • name (String)
