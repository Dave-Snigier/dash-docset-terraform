awscc_paymentcryptography_key (Resource)

Definition of AWS::PaymentCryptography::Key Resource Type

Example Usage

Generating a 2KEY TDES key

resource "awscc_paymentcryptography_key" "example" {
  exportable = true
  enabled    = true
  key_attributes = {
    key_algorithm = "TDES_2KEY"
    key_class     = "SYMMETRIC_KEY"
    key_modes_of_use = {
      verify   = true
      generate = true
    }
    key_usage = "TR31_C0_CARD_VERIFICATION_KEY"
  }

  tags = [{
    key   = "Modified By"
    value = "AWSCC"
  }]

}

Generating a Pin Encryption Key

resource "awscc_paymentcryptography_key" "example" {
  exportable = true
  enabled    = true
  key_attributes = {
    key_algorithm = "TDES_3KEY"
    key_class     = "SYMMETRIC_KEY"
    key_modes_of_use = {
      encrypt = true
      decrypt = true
      wrap    = true
      unwrap  = true
    }
    key_usage = "TR31_P0_PIN_ENCRYPTION_KEY"
  }

  tags = [{
    key   = "Modified By"
    value = "AWSCC"
  }]

}

  }]

}

Schema

Required

  • exportable (Boolean)
  • key_attributes (Attributes) (see below for nested schema)

Optional

  • enabled (Boolean)
  • key_check_value_algorithm (String)
  • tags (Attributes List) (see below for nested schema)

Read-Only

  • id (String) Uniquely identifies the resource.
  • key_identifier (String)
