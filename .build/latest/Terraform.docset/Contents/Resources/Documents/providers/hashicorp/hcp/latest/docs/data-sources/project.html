hcp_project (Data Source)

The project data source retrieves the given HCP project.

Example Usage

data "hcp_project" "example" {
  project = var.project_id
}

Schema

Optional

  • project (String) The id of the project. May be given as "" or "project/". If not set, the provider project is used.

Read-Only

  • description (String) The project's description
  • name (String) The project's name.
  • resource_id (String) The project's unique identifier
  • resource_name (String) The project's resource name in format "project/"
