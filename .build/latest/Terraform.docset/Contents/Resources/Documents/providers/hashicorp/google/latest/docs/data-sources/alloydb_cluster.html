google_alloydb_cluster

Use this data source to get information about the available cluster. For more details refer the API docs.

Example Usage

data "google_alloydb_cluster" "qa" {
}

Argument Reference

The following arguments are supported:

  • cluster_id - (Required) The ID of the alloydb cluster that the instance belongs to. 'alloydb_cluster_id'

  • project - (optional) The ID of the project in which the resource belongs. If it is not provided, the provider project is used.

  • location - (optional) The canonical id of the location.If it is not provided, the provider project is used. For example: us-east1.

Attributes Reference

See google_alloydb_cluster resource for details of all the available attributes.
