This datasource is in beta, and should be used with the terraform-provider-google-beta provider. See Provider Versions for more details on beta datasources.

Example Usage

data "google_runtimeconfig_config" "run-service" {
  name = "my-service"
}

Argument Reference

The following arguments are supported:

  • name - (Required) The name of the Runtime Configurator configuration.
  • project - (Optional) The project in which the resource belongs. If it is not provided, the provider project is used.

Attributes Reference

See google_runtimeconfig_config resource for details of the available attributes.
