awscc_mediapackagev2_channel_group (Resource)

Represents a channel group that facilitates the grouping of multiple channels.

Example Usage to create a MediaPackage V2 channel group.

resource "awscc_mediapackagev2_channel_group" "example" {
  channel_group_name = "example"
  description        = "example"

  tags = [{
    key   = "Modified By"
    value = "AWSCC"
  }]
}

Schema

Required

  • channel_group_name (String)

Optional

  • description (String)

    Enter any descriptive text that helps you to identify the channel group.

  • tags (Attributes List) (see below for nested schema)

Read-Only

  • arn (String)

    The Amazon Resource Name (ARN) associated with the resource.

  • created_at (String)

    The date and time the channel group was created.

  • egress_domain (String)

    The output domain where the source stream should be sent. Integrate the domain with a downstream CDN (such as Amazon CloudFront) or playback device.

  • id (String) Uniquely identifies the resource.
  • modified_at (String)

    The date and time the channel group was modified.

Nested Schema for tags

Optional:

  • key (String)
  • value (String)

Import

Import is supported using the following syntax:

$ terraform import awscc_mediapackagev2_channel_group.example "arn"
