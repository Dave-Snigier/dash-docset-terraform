Data Source: auth0_branding

Use this data source to access information about the tenant's branding settings.

Example Usage

data "auth0_branding" "my_branding" {}

Schema

Read-Only

  • colors (List of Object) Configuration settings for colors for branding. (see below for nested schema)
  • favicon_url (String) URL for the favicon.
  • font (List of Object) Configuration settings to customize the font. (see below for nested schema)
  • id (String) The ID of this resource.
  • logo_url (String) URL of logo for branding.
  • universal_login (List of Object) Configuration settings for Universal Login. (see below for nested schema)

Nested Schema for colors

Read-Only:

  • page_background (String)
  • primary (String)

Nested Schema for font

Read-Only:

  • url (String)

Nested Schema for universal_login

Read-Only:

  • body (String)
