Data Source: auth0_branding_theme

Use this data source to access information about the tenant's branding theme settings.

Example Usage

data "auth0_branding_theme" "my_branding_theme" {}

Schema

Read-Only

  • borders (List of Object) (see below for nested schema)
  • colors (List of Object) (see below for nested schema)
  • display_name (String) The display name for the branding theme.
  • fonts (List of Object) (see below for nested schema)
  • id (String) The ID of this resource.
  • page_background (List of Object) (see below for nested schema)
  • widget (List of Object) (see below for nested schema)
