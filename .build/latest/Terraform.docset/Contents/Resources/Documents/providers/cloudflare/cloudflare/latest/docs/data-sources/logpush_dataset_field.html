cloudflare_logpush_dataset_field (Data Source)

Example Usage

data "cloudflare_logpush_dataset_field" "example_logpush_dataset_field" {
  dataset_id = "gateway_dns"
  account_id = "account_id"
  zone_id = "zone_id"
}

Schema

Required

  • dataset_id (String) Name of the dataset. A list of supported datasets can be found on the Developer Docs.

Optional

  • account_id (String) The Account ID to use for this endpoint. Mutually exclusive with the Zone ID.
  • zone_id (String) The Zone ID to use for this endpoint. Mutually exclusive with the Account ID.
