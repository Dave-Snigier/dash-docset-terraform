cloudflare_zero_trust_risk_behavior (Resource)

Example Usage

resource "cloudflare_zero_trust_risk_behavior" "example_zero_trust_risk_behavior" {
  account_id = "account_id"
  behaviors = {
    foo = {
      enabled = true
      risk_level = "low"
    }
  }
}

Schema

Required

  • account_id (String)
  • behaviors (Attributes Map) (see below for nested schema)

Nested Schema for behaviors

Required:

  • enabled (Boolean)
  • risk_level (String)
