gitlab_instance_variables (Data Source)

The gitlab_instance_variables data source allows to retrieve all instance-level CI/CD variables.

Upstream API: GitLab REST API docs

Example Usage

data "gitlab_instance_variables" "vars" {}

Schema

Read-Only

  • id (String) The ID of this resource.
  • variables (List of Object) The list of variables returned by the search (see below for nested schema)

Nested Schema for variables

Read-Only:

  • description (String)
  • key (String)
  • masked (Boolean)
  • protected (Boolean)
  • raw (Boolean)
  • value (String)
  • variable_type (String)
