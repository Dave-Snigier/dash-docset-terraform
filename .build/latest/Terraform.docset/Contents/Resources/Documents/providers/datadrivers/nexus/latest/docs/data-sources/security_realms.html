Data Source nexus_security_realms

Use this data source to list all security realms.

Example Usage

data "nexus_security_ldap" "default" {}

Schema

Read-Only

  • active (List of Object) List of active security realms (see below for nested schema)
  • available (List of Object) List of available security realms (see below for nested schema)
  • id (String) Used to identify data source at nexus

Nested Schema for active

Read-Only:

  • id (String)
  • name (String)

Nested Schema for available

Read-Only:

  • id (String)
  • name (String)
